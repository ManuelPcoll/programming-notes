# Detalles iniciales

Todo está entre paréntesis. Todo.

Lo que está entre paréntesis se llama `form`.

## Operadores

No hay. Todo son funciones. :D

## Tipos de datos

* Listas  
```clojure
(def my-list '(1 2 3 4))
```

* Vectores  
```clojure
(def my-vector [1 2 3 4])
```

* Maps  
```clojure
(def my-map {:one 1 :two 2 :tree 3 :four 4})
```

* Sets  
```clojure
(def my-set #{1 2 3 4})
```
